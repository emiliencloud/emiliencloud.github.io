Les projets ci-dessous ont été réalisé dans le cadre de mes études en Licence Pro. Pour la plupart, les livrables étaient attendues rapidement, avec des délais bien trop court pour ce type de projets, comparé au délais demandés en entreprise...

• My Study Room

Le principe est simple créer l'interface d'une application de recherche de collocation. Mon groupe et moi, étions chargés de crééer la page de recherche. Notre premiere étape a donc été de réaliser le moodboard afin de choisir les couleurs et les typographie dans l'esprit de la collocation.

• WaterPlace

Un complexe aquatique fictif, souhaitant s'implanter dans Dijon nous à contacté dans le cadre de notre Licence Pro.
Le but : Concevoir un site web permettant de promouvoir le complexe ainsi qu'un plan de communication complet (Vidéo, réseaux sociaux, affiches, etc).
